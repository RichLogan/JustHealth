Style¶

public class Style¶

Android doesn’t have a method to programmatically apply styles, so rolled our own

Methods¶

styleButton¶

public static void styleButton(Button b, String type, LinearLayout ll, Context c)¶

Most of the time we’re programmatically generating buttons, so this allows use to apply our primary/success/warning/danger styles.

Parameters:
  • b – The button to style
  • type – The style to apply (primary/success/warning/danger)
  • ll – The linear layout the button will be applied to
  • c – getApplicationContext()
